# Career Crox CRM Blueprint

This document is included so the `/blueprint` page always opens without errors.

## Core areas
- Dashboard
- Candidates
- Interviews
- Submissions
- Tasks
- Reports
- Approvals
- Notifications
- Team Chat
- Learning Hub
- Performance Centre
- Client Pipeline
- Revenue Hub
- Recent Activity

## Role access
- Recruiter: daily execution views
- TL: team performance and approvals
- Manager: business, revenue and client controls

## Security controls
- Single active session per user
- Recruiter phone masking
- Manager and TL full phone visibility
- Activity logging
- Role-based visibility

## Deployment
- Local run with SQLite demo database
- Supabase SQL files available in `/docs`
- Render deployment steps available in the project docs
