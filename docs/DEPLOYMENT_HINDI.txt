1) Supabase me naya project banao.
2) SQL Editor me docs/SUPABASE_SCHEMA.sql run karo.
3) Is project ka code GitHub par daalo.
4) Render me New Web Service lo.
5) Repo connect karo.
6) Build command: npm install && npm run build
7) Start command: npm start
8) Environment variables add karo:
   - SUPABASE_URL
   - SUPABASE_SERVICE_ROLE_KEY
   - JWT_SECRET
   - PORT=10000
9) Deploy hone ke baad site kholo.
10) First-time setup screen par manager account banao.
