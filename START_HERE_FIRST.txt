Career Crox CRM - simple local run

Do NOT run app.py by clicking it directly in VS Code terminal.
Use one of these:

Option 1 (easiest)
- Double click RUN_CRM.bat

Option 2 (manual)
- Open terminal in this folder
- Run: py -m pip install -r requirements.txt
- Run: py app.py
- Open: http://127.0.0.1:5000
